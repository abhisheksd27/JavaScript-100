1.What is a loop? what are the types of loops in js?
A loop is a programming way to run a piece of code repeatedly until a certain condition is met


2.What is the difference between while and for loop? when we use it?
A while loop executes a block of code as long as a specified condition is true
A for loop is used to create loops that run a specific number of times


3. How do you break and continue a loop in JavaScript?
A loop can be broken using the break statement
A loop can be continued using the continue statement


4.difference between while and do-while loop?
A while loop executes a block of code as long as a specified condition is true
A do-while loop executes a block of code once, then checks a specified condition before repeating the block


5.What is the difference between for and for...of loop in js?
A for loop is used to create loops that run a specific number of times
A for...of loop is used to iterate over iterable objects like arrays, strings, etc.


6.what is the difference between forOf and ForIn loops in js?
for...of loop is used to loop through the values of an object like arrays and strings
it allows you to access each value directly without having to use and index.

for...in loop is used to loop through the properties of an object.
it allows u to iterate over trhe keys of an object and access the values associated by using keys as an index.

forEach method is used to iterate over each element in an array or object

forEach method is suitable when you want to perform an action on each element of an array without using a loop.e continue or break statement here...

9. How to use map, filter and reduce methods in JavaScript?
map method is used to create a new array with the results of calling a provided function on every element in the calling array.
filter method is used to create a new array with all elements that pass the test implemented by the provided function.
reduce method is used to reduce an array to a single value (the initialValue if provided; else, the first element in the array).
