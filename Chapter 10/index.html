Ans:

9. what is the difference between deep copy and shallow copy in js?
Ans:
- Deep copy creates a completely independent copy of an object, including its nested objects and arrays. It recursively copies all the properties and values, including any nested objects or arrays..
- Shallow copy creates a new object and only copies the reference to the original object's properties and values. It does not create a completely independent copy of the object, including any nested objects or arrays.


10. What is Set Objects in JS?
- Set objects are similar to arrays, but they only allow unique values. They are created using the Set() constructor or the new keyword.
- Set objects have methods like add(), delete(), has(), and clear().
