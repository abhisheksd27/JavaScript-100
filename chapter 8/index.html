1. What is DOM ? what is the difference between DOm and HTML?
DOM stands for Document Object Model. It is a programming interface for HTML and XML documents.
It defines the logical structure of documents and the way a document is accessed and manipulated.
Difference between DOM and HTML:
- DOM is a programming interface for HTML and XML documents.
- HTML is a markup language for creating web pages.


2. how do you select, modify or remove DOM elements?
- Selecting elements: You can use the `getElementById()`, `getElementsByClassName()`, `getElementsByTagName()`, `querySelector()`, and `querySelectorAll()` methods to select elements in a document.
- Modifying elements: You can modify the content, attributes, styles, and other properties of elements using JavaScript.
- Removing elements: You can remove elements from the DOM using the `removeChild()` method.

3. What are selectors in JS?
Selectors are used to select HTML elements based on their attributes, class names, or other criteria.

4. what is the difference between getElementById()`, `getElementsByClassName()`, `getElementsByTagName()`, `querySelector()`, and `querySelectorAll()`
- `getElementById()`: Selects the element with the specified ID.
- `getElementsByClassName()`: Selects all elements with the specified class name.
- `getElementsByTagName()`: Selects all elements with the specified tag name.
- `querySelector()`: Selects the first element that matches the specified selector.
- `querySelectorAll()`: Selects all elements that match the specified selector.


5. what are the methods to modify elements properties and attributes?
- `setAttribute()`: Sets the value of an attribute on an element.
- `removeAttribute()`: Removes an attribute from an element.
- `innerHTML`: Sets or gets the HTML content of an element.
- `textContent`: Sets or gets the text content of an element.
- `style`: Allows you to access and modify the inline styles of an element.



6. what is the difference between innerHTML and textContent
- `innerHTML`: Gets or sets the HTML content of an element. It interprets HTML tags and scripts.
- `textContent`: Gets or sets the text content of an element. It does not interpret HTML tags and scripts.

7. how to add or remove properties of HTMl eemnts in the dom?
- You can add or remove properties of HTML elements in the DOM using JavaScript. For example, you can add a new attribute to an element using `setAttribute()` and remove an attribute using `removeAttribute()`.


8. how to add or remove style of html elements in the dom isng JS?
- You can add or remove styles of HTML elements in the DOM using JavaScript. For example, you can add a new style using `style.property = value` and remove a style using `style.property = ""`.


9. How to create new elements in DOm using JS? What is the difference between createElement() and CloneNode()?
- `createElement()`: Creates a new element with the specified tag name.

- `cloneNode()`: Creates a copy of an existing node (including its event listeners and child nodes).


10. what is the difference between createElement and createTextNode in JS?
- `createElement()`: Creates a new element with the specified tag name.
- `createTextNode()`: Creates a new Text node with the specified text content.
