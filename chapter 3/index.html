1.What are operators? what are the types of operators?
Operators are used to perform specific operations on variables and values. There are two types of operators:
a. Arithmetic operators: +, -, *, /, %
b. Assignment operators: =, +=, -=, *=, /=, %=, **=

2.What is operator precedence?
Operator precedence determines the order in which operators are evaluated when there are multiple operators with the same precedence.
BODMAS rule


3. what is the difference between unary , binary and turnary operators
Unary operators :
a. Increment and decrement operators (++, --)
b. Unary plus (+) and unary minus (-)
Binary operators :
a. Arithmetic operators (+, -, *, /, %, **)
b. Assignment operators (=, +=, -=, *=, /=, %=, **=)
c. Comparison operators (==,!=, >, <, >=, <=)
d. Logical operators (&&, ||,!)
e. Bitwise operators (&, |, ^, ~, <<, >>)
Turnary operators :
a. Conditional (ternary) operator (condition? value1 : value2)

5. What is the difference between '==' and '==='?
'==' checks if two values are equal, but it converts the data types of both values before comparison.
'===' checks if two values are equal and have the same data type.
