2. what is the difference between find() and filter() and slice() methods of an array?
Ans:- The find() method returns the value of the first element in the provided array that satisfies the provided testing function. If no element satisfies the condition, it returns undefined.
The filter() method creates a new array with all elements that pass the test implemented by the provided function.
The slice() method returns a shallow copy of a portion of an array into a new array object selected from begin to end (end not included).

3. what is the difference between push() and concat() methods pf an array?
Ans:- The push() method adds one or more elements to the end of an array and returns the new length of the array.
The concat() method returns a new array that is the result of concatenating all of the arrays and/or values provided.

4. What is the difference between pop() and shift() methods of an array?
Ans:- The pop() method removes the last element from an array and returns that element.
The shift() method removes the first element from an array and returns that element.

5. What is the difference between splice() and slice() methods of an array?
Ans:- The splice() method changes the contents of an array by removing or replacing existing elements and/or adding new elements in place.

6. What is the difference between map() and forEach() array methods in iteration?
Ans:- The map() method creates a new array with the results of calling a provided function on every element in the calling array.
The forEach() method executes a provided function once for each array element.

7. how to sort and reverse and array?
Ans:- To sort an array in ascending order, you can use the sort() method.
To sort an array in descending order, you can use the sort() method with a custom comparison function.
To reverse an array, you can use the reverse() method.
