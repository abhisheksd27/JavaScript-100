1. what is a string?
A string is a datatype used to store and manipulate data.


2. What are template literals and string interpolation in strings?
Template literals are enclosed by backticks (`` ` ``) and allow multi-line strings and embedded expressions.
String interpolation is a feature that allows you to insert the value of a variable or expression inside a string.


3. what is the differenc between "" ,'' and `` in jS?
"" - This is a string literal that supports escape sequences.
'' - This is a string literal that does not support escape sequences.
`` - This is a template literal that supports embedded expressions.

5. What is string immutability?
String immutability means that once a string is created, its characters cannot be changed. This prevents unintended consequences when working with strings, such as performance optimizations and security concerns.
