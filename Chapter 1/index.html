3.What is scope in JS?
Ans:- Scope is the visibility and accessibility of variables, functions, and objects in a particular part of a program.
Global scope - variables defined outside of any function have global scope and can be accessed from anywhere in the program
Local scope - variables defined inside a function have local scope and can only be accessed within that function

5.What is hoisting in JS?
Ans:- Hoisting is a JavaScript mechanism where variables and function declarations are moved to the top of their scope before code execution.
let and const doesnt allow hoisting

6.What is the difference between let and const in JS?
Ans:- let and const are block scope variables.
let - let is block scoped and can be reassigned.
const - const is block scoped and cannot be reassigned.
