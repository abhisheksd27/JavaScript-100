14. What is function currying in JS?
Ans:- Function currying is a technique of transforming a function with multiple arguments into a sequence of functions, each accepting a single argument.

Advatages:- - It makes the code more readable and easier to maintain. - It can be used to create more flexible and reusable functions. - It can help to avoid errors by ensuring that the correct arguments are passed to the function.

15. what are call , apply and bind in JS?
Ans:- Call, apply and bind are methods of Function.call(), Function.apply() and Function.bind() respectively.
Call method: This method calls a function with a given this value and arguments provided individually.

Apply method: This method calls a function with a given this value and arguments provided as an array.

Bind method: This method creates a new function that calls the specified function with a given this value and arguments provided individually. The this value is fixed to the specified value, and any additional arguments are provided as arguments to the new function.
